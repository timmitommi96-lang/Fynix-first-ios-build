# Fynix V2 Analyse

## V1 Bestandsaufnahme
Die V1 ist eine einzelne HTML-Datei mit eingebetteten Base64-Bildern (IMGS-Objekt).
Sie nutzt Puter.com API für Backend und Anthropic API für AI-Chat.

### Vorhandene Features in V1:
- Login/Register (E-Mail-basiert, localStorage)
- Onboarding (4 Schritte: Name, Klasse, Lernstil, Avatar)
- Home Screen (XP-Karte, Streak, Vokabeln)
- KI Tools (Lynx Chat, Quiz Generator, Lernhilfe, Lernzettel)
- Vokabel-System (Erstellen, Foto-Scan, Text-Extraktion, Abfrage)
- Ranking (statische Daten)
- Profil
- Bottom Navigation

### Fehlende Features laut PRD (für V2):
1. **Splash Screen** - Fehlt komplett
2. **Endless Learning Feed** (TikTok-Style) - KOMPLETT FEHLT
3. **Swipe-to-Quiz Mechanik** - KOMPLETT FEHLT
4. **Detailliertes XP/Level-System** mit Roast - Teilweise vorhanden
5. **Streak-System mit Boni** (+5%, +10%, +50% XP) - Basis vorhanden, Boni fehlen
6. **Maskottchen-System** mit Emotionen - Nur statisch, keine Emotionen
7. **Roast- & Feedback-System** - KOMPLETT FEHLT
8. **Habit-Tracker** (gamified) - KOMPLETT FEHLT
9. **Money-Tracker** - KOMPLETT FEHLT
10. **Joker & Truhen** - KOMPLETT FEHLT
11. **Detailliertes Onboarding** (7 Fragen statt 4) - Teilweise
12. **Personalisierungsfragen** (Schule, Motivation, Gewohnheiten, Tonfall, 30-Tage-Ziel) - FEHLT
